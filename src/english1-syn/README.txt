Linux
-----

Start of the syntax project:

Please type:

./jslim.sh 

JSLIM starts automatically.

If you want to switch to LA-Morph, please type -m after the start of JSLIM.


Windows
-------

Start of the syntax project:

Please type:

jslim.bat

JSLIM starts automatically.

If you want to switch to LA-Morph, please type -m after the start of JSLIM.
